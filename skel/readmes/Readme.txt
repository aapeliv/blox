About
=====

A small game where you form rows out of dropping blocks.



Installation
============

To install AapeliBlox, simply extract AapeliBlox.zip into some directory and run
AapeliBlox.exe from there. If you get errors about .dll files, make sure you
have extracted the file in a folder along with all the appropriate .dll files.
